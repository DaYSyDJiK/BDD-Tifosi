# Base de données Tifosi

## Objectif
Conception et implémentation d'une base de données MySQL pour le restaurant de street-food italien Tifosi.

## Structure du projet

- 00_setup_user.sql → Création de l'utilisateur MySQL
- 01_schema.sql → Création de la base et des tables
- 02_insert_test_data.sql → Insertion des données de test
- 03_test_queries.sql → Requêtes de vérification

## Technologies utilisées

- MySQL
- phpMyAdmin

## Contenu de la base

Tables principales :
- focaccia
- ingredient
- boisson
- marque
- menu
- client

Tables de liaison :
- comprend
- contient
- achat

Les 10 requêtes demandées dans le brief ont été implémentées et testées
